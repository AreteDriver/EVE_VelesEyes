# Screenshot Instructions for README

## Main Screenshot (screenshot.png)

Capture a screenshot showing:

1. **Main Window** - The Argus Overview application with multiple tabs visible
2. **2x2 Grid Layout** - 4 EVE client previews arranged in a grid
3. **Characters & Teams Tab** - Show a fleet composition with roles assigned
4. **Alert Border** - One preview with a red border (simulating damage alert)
5. **Layout Dropdown** - The preset selector showing saved configurations

### Recommended Setup:
- Resolution: 1920x1080 or higher
- 4 EVE clients logged in (or use placeholder windows titled "EVE - CharacterName")
- Dark theme enabled
- Show the full application window including title bar

### File Specifications:
- Format: PNG
- Name: `screenshot.png`
- Location: `docs/screenshot.png`

## Demo GIF (optional: demo.gif)

Record a short GIF (5-10 seconds) demonstrating:

1. Click on a preview to switch active window
2. Use Ctrl+Tab to cycle through windows
3. Show the visual feedback on click

### Tools for Recording:
- `peek` - Simple screen recorder for Linux
- `byzanz` - Command-line GIF recorder
- `gifski` - High-quality GIF encoder

### Example with peek:
```bash
sudo apt install peek
peek  # Select region and record
```

### File Specifications:
- Format: GIF
- Max size: 5MB
- Name: `demo.gif`
- Location: `docs/demo.gif`
